# 하루니(Haruni)
매일 일기를 쓰는거 대신, **하루니**와의 대화를 통해 일상을 기록해보세요! <br><br>
하루니는 당신의 **친구**가 될 수도, **가족**이 될 수도, **맨토**가 될 수도 있습니다.<br><br>
당신을 위한 인공지능 모델 **하루니**와 함께 동반 성장의 기회를 잡아보세요!<br><br>

<br>

- AI : 송유찬, 박상원
